# Database Setup

## Required Packages

```json
{
  "dependencies": {
    "drizzle-orm": "^0.39.2",
    "better-sqlite3": "^11.8.1",
    "zod": "^3.24.1"
  },
  "devDependencies": {
    "drizzle-kit": "^0.30.4",
    "@types/better-sqlite3": "^7.6.12"
  }
}
```

## Installation

First, install the required dependencies:

```bash
# Core dependencies
pnpm add drizzle-orm better-sqlite3 zod

# Development dependencies
pnpm add -D drizzle-kit @types/better-sqlite3
```

## Configuration

### Database Client

```typescript
// src/server/db/client.ts
import { drizzle } from "drizzle-orm/better-sqlite3";
import Database from "better-sqlite3";
import * as schema from "./schemas";

const sqlite = new Database(env.DATABASE_URL);
export const db = drizzle(sqlite, { schema });
```

### Drizzle Config

```typescript
// drizzle.config.ts
import type { Config } from "drizzle-kit";

export default {
  schema: "./src/server/db/schemas/index.ts",
  out: "./src/server/db/migrations",
  driver: "better-sqlite",
  dbCredentials: {
    url: env.DATABASE_URL,
  },
} satisfies Config;
```

### Environment Setup

```bash
# .env
DATABASE_URL="sqlite:./local.db"

# .env.example
DATABASE_URL="sqlite:./local.db"
```

## Initial Setup

1. **Create Database Directory Structure**
   ```bash
   mkdir -p src/server/db/{schemas,migrations}
   ```

2. **Configure Drizzle**
   ```typescript
   // drizzle.config.ts
   import config from './src/server/db/config';
   export default config;
   ```

3. **Add NPM Scripts**
   ```json
   {
     "scripts": {
       "db:generate": "drizzle-kit generate:sqlite",
       "db:push": "drizzle-kit push:sqlite",
       "db:studio": "drizzle-kit studio"
     }
   }
   ```

## Type Safety

1. **Configure Path Aliases**
   ```json
   // tsconfig.json
   {
     "compilerOptions": {
       "baseUrl": ".",
       "paths": {
         "@/*": ["./src/*"],
         "@modules/*": ["src/modules/*"],
         "@server/*": ["src/server/*"]
       }
     }
   }
   ```

2. **Configure Next.js**
   ```javascript
   // next.config.js
   const nextConfig = {
     webpack: (config) => {
       config.resolve.alias = {
         ...config.resolve.alias,
         "@modules": "./src/modules",
         "@server": "./src/server",
       };
       return config;
     },
   };
   ```

## Usage Examples

### Creating Records
```typescript
import { db } from "@server/db/client";
import { users } from "@modules/auth/models/z.user";

const newUser = await db.insert(users).values({
  email: "user@example.com",
  name: "Test User",
  createdAt: Date.now(),
  updatedAt: Date.now(),
});
```

### Querying Records
```typescript
import { eq } from "drizzle-orm";

const user = await db.query.users.findFirst({
  where: eq(users.email, "user@example.com"),
  with: {
    profile: true,
    oauthAccounts: true,
  },
});
```

### Updating Records
```typescript
const updated = await db
  .update(users)
  .set({ name: "New Name" })
  .where(eq(users.id, userId));
```

### Deleting Records
```typescript
const deleted = await db
  .delete(users)
  .where(eq(users.id, userId));
```

## Best Practices

1. **Always Use Types**
   ```typescript
   import { InferModel } from "drizzle-orm";
   type User = InferModel<typeof users>;
   type NewUser = InferModel<typeof users, "insert">;
   ```

2. **Handle Timestamps**
   ```typescript
   const now = Date.now();
   const user = await db.insert(users).values({
     ...userData,
     createdAt: now,
     updatedAt: now,
   });
   ```

3. **Use Transactions**
   ```typescript
   await db.transaction(async (tx) => {
     const user = await tx.insert(users)...;
     await tx.insert(profiles)...;
   });
   ```

4. **Validate Input**
   ```typescript
   const userSchema = z.object({
     email: z.string().email(),
     name: z.string().min(1),
   });

   const validated = userSchema.parse(input);
   ```
